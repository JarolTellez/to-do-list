Aplicacion web para gestionar tareas personales, incluye autenticacion segura con JWT, API REST construida con Node.js y Express y una interfaz hecha con React.

# Descripción general

El objetivo es ofrecer una herramienta simple para organizar tareas diarias.  
Fue desarrollada como parte de mi portafolio para demostrar habilidades en desarrollo **full stack**, manejo de sesiones seguras y despliegue.

## Características principales

- Autenticación con **JWT + Cookies HttpOnly**
- CRUD completo de tareas
- API REST construida con **Express **
- Interfaz moderna y **responsive** (React )
- Protección de rutas y sesiones persistentes

##  Tecnologías utilizadas

### Frontend
- React  
- Vite  

### Backend
- Node.js  
- Express  
- Prisma ORM  
- MySQL  
- JWT  

### Herramientas
- Postman  
- Git & GitHub

## Instalación y configuración

### 1. Clonar el repositorio
git clone 
### 2. Instalar dependencias
npm install
### 3. Configurar variables de entorno
BACKEND_URL
EXP_ACCESS_TOKEN
EXP_REFRESH_TOKEN
FRONTEND_URL
JWT_ACCESS_SECRET
JWT_REFRESH_SECRET
MAX_SESIONES_ACTIVAS
MYSQL_URL
NODE_ENV
RAILWAY_BUILD_COMMAND
### 4. Ejecutar el servidor backend
node server.js
### 5. Iniciar el frontedn
cd Cliente
npm install
npm run dev
